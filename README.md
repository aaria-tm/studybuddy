# studybuddy

## Overview

Study Buddy Pro+ is a simple productivity web app designed to help students manage focused study sessions using timed intervals, task tracking, and break scheduling.

## Features

* Custom study timers (30, 60, 90, 120, 180 minutes)
* Break time configuration
* Pause and reset controls
* Task list for daily planning
* Dark and light mode toggle
* Sound alerts for session and break completion
* Mobile-friendly vertical scroll interface

## How It Works

Users select a study duration, set a break time, and start a session. The app runs a timer-based workflow where study sessions and breaks alternate automatically. Tasks can be added and removed to support daily planning.

## Tech Stack

* HTML
* CSS
* JavaScript (Vanilla)
* Chart.js (removed in final version if not used)

## Setup Instructions

1. Download or clone the repository
2. Open `index.html` in any modern browser
3. No additional installation required
